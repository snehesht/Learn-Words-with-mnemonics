Learn GRE Words with Mnemonics 

Requirements
************ 
		ChromeDriver for Chrome Browser
		Python 3.4.x
		AdblockPlus
		PowerMenu


Install 
*******

	Install Python 3.4.1 ( https://www.python.org/downloads ) 

	Download ChromeDriver from http://chromedriver.storage.googleapis.com/index.html and select the latest version

	Unzip the chromedriver_xxxxx.zip and copy the chromedriver.exe to Scripts folder in Python Installation folder

	Download PowerMenu from http://www.abstractpath.com/powermenu and run it

	Go to PowerMenu installation folder in your command prompt and then type

				powermenu -topmost "cmd"


Run
***
	
	Run words.py in the terminal

				python words.py
